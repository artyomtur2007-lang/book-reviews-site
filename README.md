## **FilmBook Reviews**

Веб-приложение на Django для просмотра, поиска и оценки книг.
Пользователи могут регистрироваться, оставлять отзывы и выставлять оценки, на основе которых формируется рейтинг книг.

## **Возможности проекта**

Регистрация и авторизация пользователей
Просмотр каталога книг
Поиск книг по названию
Фильтрация по году и жанрам
Пагинация каталога
Добавление книг авторизованными пользователями
Добавление отзывов и оценок к книгам
Автоматический подсчёт среднего рейтинга книги
Отображение ТОП-10 книг по рейтингу на главной странице

## **Установка и запуск проекта**
1. Клонировать репозиторий:
git clone https://github.com/artyomtur2007-lang/filmbook-reviews.git
cd filmbook-reviews
2. Создать и активировать виртуальное окружение:
python -m venv .venv
source .venv/bin/activate (Linux / macOS)
.venv\Scripts\activate (Windows)
3. Установить зависимости:
pip install -r requirements.txt
4. Применить миграции:
python manage.py migrate
5. (Опционально) заполнить базу тестовыми данными:
python manage.py seed_books
6. Запустить сервер:
python manage.py runserver
После этого сайт будет доступен по адресу
http://127.0.0.1:8000/
## **Запуск тестов**
Для проверки корректности работы проекта используются встроенные Django-тесты.
Запуск всех тестов:
python manage.py test
Покрываются следующие сценарии:
регистрация и вход пользователей
CRUD-операции
поиск и фильтрация
пагинация
